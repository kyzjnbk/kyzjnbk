!!! caution "++ctrl+s++"

    有时会习惯性地写完程序后按下++ctrl+s++，潜意识地认为这是“保存”
    反应过来后发觉不对，又开始按esc，结果发现按啥都没反应了
    查找资料后得知，
    这是因为++ctrl+s++的组合快捷键在vim中是“阻塞后续输入”的作用
    可以按++ctrl+q++以取消。
