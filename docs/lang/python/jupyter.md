# Jupyter Notebook

交互式、笔记本式的编程环境。

[![try jupyter notebook](jupyter.asserts/trynb.png)](https://jupyter.org/try-jupyter/retro/notebooks/?path=notebooks/Intro.ipynb)

## Quick Start

```bash
# 使用pip安装
pip install notebook

# 启动 Jupyter Notebook
jupyter notebook
```
